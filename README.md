### Features
- Data fetched directly from Yahoo Finance using `yfinance`.

- ML models used:
  - Logistic Regression
  - Random Forest (comparison)
- Model evaluation using accuracy, confusion matrix, and simple backtesting.
- Visualizations: stock price trends, prediction accuracy, comparison of model performance.
